# Icon Color Palette

Shared color palette used across all feature icon animations (Lottie + SVG).

## Primary Colors

| Color         | Hex       | Usage                                      |
|---------------|-----------|---------------------------------------------|
| Deep Blue     | `#0857A0` | Primary fill, strokes, borders              |
| Bright Blue   | `#2294E6` | Highlights, secondary fills                 |
| Sky Blue      | `#2D99FF` | Accents, lighter elements                   |
| Teal          | `#00DDB3` | Key accent (lock, chart highlights)         |
| Mint          | `#27F6D0` | Light accent variation                      |

## Supporting Colors

| Color         | Hex       | Usage                                      |
|---------------|-----------|---------------------------------------------|
| Dark Navy     | `#0A264C` | Dark backgrounds, deep shadows              |
| Mid Navy      | `#0A3966` | Secondary dark fills                        |
| Steel Blue    | `#325A7F` | Muted elements, secondary strokes           |
| Slate         | `#4E667D` | Tertiary fills                              |
| Light Gray    | `#D9DFE6` | Borders, subtle fills                       |
| Off White     | `#F4F4F4` | Light backgrounds                           |
